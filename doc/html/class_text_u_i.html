◆ setTimer()

void TextUI::setTimer ( uiTimer_t  msec)

Set timer interval in milliseconds.

A value > 0 enables events of type EVENT_TYPE_TIMER. Every 'msec' milliseconds such an event is generated. A value of 0 disables timer events. A value of less that 10 milliseconds is rounded to 10 milliseconds.

Parameters
msecThe new timer interval in milliseconds.
