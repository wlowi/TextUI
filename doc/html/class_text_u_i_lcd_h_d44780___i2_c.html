Detailed Description

A driver for OLED displays with HD44780 and similar controller.

Example:

#include "TextUI.h" #include "TextUILcdHD44780_I2C.h"

/o Raspberry RP 2040 support multiple pin configurations o/ Wire.setSDA(PIN_LCD_SDA); Wire.setSCL(PIN_LCD_SCL);

textUI.setDisplay( new TextUILcdHD44780_I2C( &Wire, 0x27 /o i2c addr o/ );
