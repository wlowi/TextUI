Detailed Description

A proxy driver that supports both TextUIInput, TextUILcd.

This class supports communication with an external user terminal. It replaces both display and input classes of TextUI.

Example: 

#include "TextUI.h"
#include "TextUIStreamProxy.h"
#include "HomeScreen.h"

TextUI textUI;

void setup()
{
  Serial1.begin(19200, SERIAL_8N1);

  TextUIStreamProxy *streamProxy = new TextUIStreamProxy( Serial1);

  textUI.setDisplay( streamProxy);
  textUI.setInput( streamProxy);
  textUI.setHomeScreen( new HomeScreen());
}
